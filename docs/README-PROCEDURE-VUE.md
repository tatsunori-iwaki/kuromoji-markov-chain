
# setup
```
cd ~/IdeaProjects
vue create kuromoji-markov-chain
```

# run
```
cd ~/IdeaProjects/kuromoji-markov-chain
yarn serve
```

# modules
```
cd ~/IdeaProjects/kuromoji-markov-chain
yarn add gh-pages
yarn add kuromoji zlibjs
yarn add bootstrap-vue
```

# gh-pages
```
cd ~/IdeaProjects/kuromoji-markov-chain
yarn build
yarn deploy
```
